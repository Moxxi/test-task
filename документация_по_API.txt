Функционал для администратора системы:

/admin/auth -GET - авторизация в системе - возвращает токен для использования функицонала администратора.
/admin/poll/create – POST – {Name : строка, DateStart: cтрока, D:M:Y,DateEnd: cтрока, D:M:Y,Description: строка,} – создает опрос в системе
/admin/poll/delete – POST – {Name: строка} – удаляет опрос с указанным именем.
/admin/poll/change – POST – {Name: строка, Change: {Name: “...”, DateEnd: “...“, Description: "..." }} – изменяет данные опроса, которые указаны в словаре Change.
/admin/question/create – POST – {Name – название опроса, которому будет добавлен вопрос, Text: строка, Type: варианты выбора [TX - text, OC – one choice ,MC – many choice], Options: {“first” : “100 лет ”,“second”: “150 лет”} - словарь вариантами ответа, если тип text, то NULL } – создает вопрос в системе.
/admin/question/delete – POST – {Text: строка} – удаляет опрос с указанным текстом.
/admin/question/change – POST – {Text: строка, Change: { Text: “ ”, Type: “ “, Options: ” ” }} – изменяет данные вопроса, которые указаны в словаре Change.


Функционал для пользователей системы:

/user/polls/get – GET – возвращает список активных опросов (только название и описание).
/user/poll/get – POST – возвращает один опрос подробно, с вопросами.
/user/polls – POST  {Id : id пользователя} - возвращает список опросов пройдённых пользователем (только название и описание).
/user/poll – POST - {Name: “...”, Id: Id пользователя} – возвращает один опрос пройдённый пользователем, подробно со списком вопросов и ответов.
/user/answer/create  - POST – { Name: название опроса, Id – id пользователя, Answer : {‘Text1”: “answer1”, ‘Text2”: “answer2”, ‘Text3”: “answer3”} – словарь с вопросом – ответом.} – записывает ответы пользователя, для указанного опроса.
